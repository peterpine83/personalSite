Coach!

Zombie Diner

This is a single page website that the owner of the Zombie Diner, Rick, can use to keep track of the dishes offered at the diner. Rick can see all dishes, add dishes, delete dishes, and update dishes. When dishes are created a card is produced that will show an image of the dish, the name of the dish, and the price. There are edit and delete buttons on each card for easy removal or update of the dish. The input form has a validation element to it so that the Rick cannot enter invalid information. The site is also responsive and will adjust to screen size.

Technologies Used
  • Express
  • SQlite3
  • Backbone.js
  • SemanticUI
  • GitHub: Zombie Diner Repository
  • Date: April 2015
  • Link to App: Zombie Diner
